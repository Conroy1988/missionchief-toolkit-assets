# UK Mission Requirement Capability Port

This work expands the Mission Requirements Matrix with the complete current UK vehicle and personnel capability dataset used for MissionChief requirement reconciliation.

## Scope

- Preserve the existing Matrix interface and selected/responding/on-site lifecycle.
- Add the complete UK requirement alias and vehicle-type capability table.
- Prioritise longer requirement phrases before shorter generic aliases.
- Keep unknown requirement text visible.
- Keep all capability data bundled locally with no new runtime host.
- Add dataset-driven parser and vehicle-type regression coverage.

## Release target

Toolkit v4.20.6.

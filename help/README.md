# MissionChief Map Command Toolkit Help Centre

Public user documentation for the MissionChief Map Command Toolkit.

- Main guide: `help/index.html`
- Search is client-side and self-contained.
- The userscript fetches the guide from `raw.githubusercontent.com` and renders it inside a sandboxed browser-style Help Centre.
- No MissionChief account data is sent to this repository.
- Major Toolkit releases must update the guide when user-facing behaviour changes.

Current Toolkit coverage: **v4.2.0**
